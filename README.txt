This is the files that were used in the project GoFlow° at Chalmers University of Technology.

The app is available on the App Store under the name GoFlow°.

- "Balance Ball" contains the Arduino files that were programmed for the custom made ball.
- "GoFlow_Testserver_Random" contains a Java program (made in Processing) to test the app with random values.
- "iOS App" contains all the app files for XCode programmed in Swift 2.
- "KinectProgram" contains the Java program (made in Processing) to run exercises with the Kinect (v2 Xbox One) camera.
